Keyboard and Touch navigation

Obviously you can use the commands in the toolbar above in order to control the slideshow. But for the most things you can also use the keyboard or touch gestures.

Keyboard Shortcuts

Left Arrow      Go to previous slide
Right Arrow      Go to next slide
Space      Go to next slide
Enter      Go to next slide
ESC      Show table of contents
1      Show table of contents
2      Show slides
3      Show print view
P      Toggle presentation mode
B      Fade out to black
W      Fade out to white

Touch Gestures

Swipe Left      Go to next slide
Swipe Right      Go to previous slide
Double Tap      Toggle presentation mode
Long Press      Toggle slide overview

Getting Started

Slides are simple HTML. In order to get started …

  • Copy this template project
  • Edit the file static/index.html
  • Build the presentation

In order to build the presentation run any of these:

  • For development: npm run devserver
  • For deployment: npm run build

The last command will run a local webserver on port 8080. Open http://localhost:8080 to access it.

Getting started with lecture-slide.js is easy. At least if you know a little bit HTML and if you are not afraid of running a command-line tool on your computer. Take this presentation, for example. It is contained in a single HTML file. And all it takes to build the final presentation is a simple command in your operating system's command line.

But first things first. In order to get started you need to create a little structure on your computer. This is you need to create a directory which must contain a certain set of given files. Basically there are many ways, how this can be done and each has its own merits. But here I show you a way which is both simple and yet powerful. This means it will grow with your needs as you keep going on.

Getting the template project

There is already a ready-to-run template project on Github. So to start, visit the link and download its content. This can be done with the green Clone or download button.

Clone or download » Download ZIP

Extract the ZIP file anywhere on your computer. That's all. So let's take a look, what to do next.

Writing your slides

Now it is time to write your slides, which are just plain HTML files in the static directory. So go there and open up the file static/index.html in any code editor of your liking. There you will see the code if this presentation.

After you made yourself comfortable with the file content, start hacking right away. Create as many <section> elements as you wish. Each one will become a slide. And don't forget to put the following content in it:

  • <h1>: This is the slide title
  • <article>: This is the slide itself
  • <aside>: This is the detail explanation you are currently reading.

Each one of these is optional. But when they are all present the final result should look something like this:

<section>
     <h1>Title</h1>
     <article>Slide content</article>
     <aside>Detailed explanation</aside>
</section>

Building and running the presentation

Now open a terminal window and go to the directory where your presentation lives. When you are there, run one of the following two commands, depending on what you want to do:

  • For development: npm run devserver
  • For deployment: npm run build

Use the first command while you're still writing your slides. It will start a local webserver which can be accessed at http://localhost:8080. When you made changes to your slides simply press F5 on or two times to reload the page.

Once your done, run the second command. It will create the final version ready to copy onto any webserver. The files will be in the build directory. Just push them on a webserver and call it a day. It really doesn't have to be a special server. Even the most simple one will do.

Translating the User Interface

This presentation is in English. So it makes sense that the buttons to navigate from slide to slide are English, too. If you want to change that:

  1. Edit the file index.js
  2. Translate the label… values there

For example in German:

import SlideshowPlayer from "lecture-slides.js";

window.addEventListener("load", () => {
    let player = new SlideshowPlayer({
        labelPrev: "Zurück",
        labelNext: "Weiter",
        labelGoTo: "Gehe zu",
        labelOverview: "Übersicht",
        labelPresentationMode: "Präsentationsmodus",
        labelSlide: "Folie",
        mode: "slideshow",
    });

    player.start();
});
